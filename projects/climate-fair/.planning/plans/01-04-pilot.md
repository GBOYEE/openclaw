# 01-04: Pilot with 50 Farmers

**Epic:** Data Ingestion & Alerts

## Goal

Deploy MVP and onboard 50 real smallholder farmers; collect usability feedback.

## Acceptance

- VPS deployed (Ubuntu + PostgreSQL + app)
- SMS alerts working end‑to‑end for at least 30 days
- Onboarded 50 farmers (via NGO partner or local enumerators)
- Feedback summary: satisfaction, message clarity, false positives
- Iteration plan for Phase 2 based on feedback

## Tasks

1. Prepare production deployment (Docker Compose or systemd)
2. Set up monitoring (logs, uptime)
3. Recruit pilot participants (through local agri‑NGO)
4. Register farmers: phone number, language, location, crop type
5. Run pilot for 4 weeks; support via WhatsApp helpdesk
6. Conduct phone surveys; analyze feedback
7. Document learnings; adjust roadmap

## Dependencies

01-03

## Time

2 weeks (including pilot run)
