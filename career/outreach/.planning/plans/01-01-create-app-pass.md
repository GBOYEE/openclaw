# 01-01: Create Gmail App Password

**Epic:** Setup

## Goal

Generate an app password for Xanderaicorp@gmail.com to use with SMTP.

## Steps

1. Go to https://myaccount.google.com/security
2. Ensure 2‑Factor Authentication is ON
3. Under "Signing in to Google" → "App passwords"
4. Choose "Mail" as app, "Other" device (or your computer name)
5. Copy the 16‑character password (spaces optional)
6. Save it somewhere safe (you'll put it in `.env`)

## Acceptance

- Have the 16‑char password copied

## Time

5 minutes
