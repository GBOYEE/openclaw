# Interview Prep Plan (02-04)

Goal: Ace technical and system design interviews for AI/automation roles.

## LeetCode (Python)

Focus: arrays, strings, recursion, graphs, sliding window, DP (basic).

Target: 50 problems in 2 weeks.

Daily: 4 problems (2 easy, 2 medium). Use Neetcode 150 list.

## System Design

Practice prompts:

1. Design an AI agent framework (agent-core)
   - Components: Agent, Planner, Memory, Tools, Reflection
   - Data flow, scaling, error handling
   - Persistence strategy, concurrency

2. Design a task scheduler (automation-engine)
   - Cron parser, job storage, misfire handling
   - Webhook server, rate limiting, auth
   - Observability: metrics, logs, alerts

3. Design a backtesting engine (trading-lab)
   - Data ingestion, indicator calculation, signal generation
   - Portfolio simulation, metrics calculation
   - Performance, memory usage

4. Design a multi-tenant SaaS (future product)
   - Tenants, isolation, billing, usage tracking
   - API keys, middleware, database schema

Deliverable: 5-minute talk + diagram (Excalidraw) for each.

## Live Demos

- agent-core: run a simple goal, show steps and reflection
- automation-engine: show dashboard, trigger task
- trading-lab: show backtest results in UI

Record 2‑minute video of each demo; keep in portfolio.

## Mock Interviews

- Use Pramp or find a peer
- 2 sessions/week until hired
