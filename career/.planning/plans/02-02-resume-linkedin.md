# 02-02: Tailor Resume & LinkedIn

**Epic**: Job Acquisition

## Goal

Craft application materials that emphasize outcomes (delivered 3 production-ready repos, GSD process, full-stack automation) rather than just duties.

## Acceptance Criteria

- Master resume (1 page) with bullet metrics: "Built agent-core v1.0.0 with 6 intelligence layers", "Delivered automation-engine with security/observability", "Created trading-lab used for backtesting"
- LinkedIn: headline "AI Automation Engineer", summary using brand statement, experience items with metrics
- Tailored versions for: embedded/infra roles vs. quant/trading roles
- Add links to portfolio, live demos

## Tasks

1. Reverse-engineer 5 job descriptions for desired roles; extract keywords
2. Rewrite bullets with CAR (Challenge, Action, Result) using numbers/stars/CI
3. Update LinkedIn; add featured projects (repos)
4. Export resume to PDF; add to portfolio site
5. Set GitHub profile to show hiring status
