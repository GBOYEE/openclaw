# 03-01: Define Consulting Offer

**Epic**: Consulting Launch

## Goal

Package AI automation expertise into a clear, fixed-scope service with pricing and delivery timeline.

## Acceptance Criteria

- Offer: "Automation Audit & Implementation" — 2-week engagement, $5k–$15k depending on scope
- Scope: audit existing processes, propose automation architecture, implement MVP using agent-core/automation-engine
- Deliverables: report, working prototype, handoff docs, 1-week support
- Statement of Work (SOW) template ready

## Tasks

1. Define tiers: Audit-only ($5k), MVP Implementation ($10k), Full Integration ($15k)
2. Document typical process (discovery call, audit, proposal, delivery, review)
3. Write SOW template with payment milestones (50% upfront, 50% on delivery)
4. Create FAQ page on website
5. Update LinkedIn & portfolio "Hire Me" page with offer details

## Dependencies

01-01, 01-02

## Time Estimate

3 hours
