# 01-01: Craft Personal Brand Statement

**Epic**: Positioning & Assets

## Goal

Define a clear, compelling "AI Automation Engineer" narrative that resonates with employers and clients.

## Acceptance Criteria

- One-sentence value proposition
- Three bullet points of key differentiators
- Bio paragraph (50 words) for LinkedIn/Twitter
- Example project snippets to illustrate impact

## Tasks

1. Analyze existing repos (agent-core, automation-engine, trading-lab) to extract standout achievements
2. Draft 3 versions of value prop; pick strongest
3. Write bio; get user feedback (quick poll)
4. Update IDENTITY.md (optional) and all GitHub profile fields

## Dependencies

None — can start immediately.

## Time Estimate

2 hours
