# 03-02: Create Sales Materials

**Epic**: Consulting Launch

## Goal

Produce one-pager and case studies to convince prospects to buy.

## Acceptance Criteria

- One-pager PDF (1 page) with: problem, solution, my expertise, process, pricing, testimonials (placeholders until first client)
- 2–3 case studies from existing projects (agent-core: "Built intelligent agent with vector memory"; automation-engine: "Secure task scheduler with retries")
- Portfolio site has dedicated "/consulting" page with CTA to book call

## Tasks

1. Write one-pager in Google Docs → export to PDF
2. Craft case studies using STAR format (Situation, Task, Action, Result)
3. Add metrics: time saved, reliability gained, scalability achieved
4. Design simple cover image (Canva)
5. Publish on site; link from LinkedIn

## Dependencies

03-01

## Time Estimate

4 hours
