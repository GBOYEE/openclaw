# 04-04: Launch Public Beta

**Epic**: Productization

## Goal

Open product to early adopters; acquire 10 paying customers.

## Acceptance Criteria

- Landing page with demo video, pricing, signup
- Beta signup form (collect email, company, use case)
- Onboarding email sequence (welcome, API key, quickstart)
- 10 paying customers within 60 days of launch
- Feedback loop: weekly surveys, Feature requests logged

## Tasks

1. Build landing page (Next.js or static HTML) on Vercel/Netlify
2. Record 5-minute demo video showing dashboard creating tasks, viewing metrics
3. Set up email (Resend) for transactional + broadcast
4. Write quickstart guide; example templates
5. Soft launch: post on Indie Hackers, Reddit r/selfhosted, Twitter; offer 50% off for first 10
6. Track MRR in Stripe; monitor support channel

## Dependencies

04-03

## Time Estimate

Ongoing; 2 weeks to launch + 8 weeks to acquire customers
