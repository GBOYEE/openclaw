# Roadmap: GBOYEE Career & Revenue

## Overview

From skills to income: get a job, land consulting clients, and launch passive products using AI automation stack.

## Phases

- [ ] **Phase 1: Positioning & Assets** — Resume, portfolio, personal brand, GitHub optimization
- [ ] **Phase 2: Job Acquisition** — Target companies, applications, interview prep, networking
- [ ] **Phase 3: Consulting Launch** — Offer template, pricing, first clients, delivery pipeline
- [ ] **Phase 4: Productization** — Turn core repos into SaaS/API services; monetize
- [ ] **Phase 5: Passive Scale** — Content, SEO, affiliate, managed services, royalties

### Phase 1: Positioning & Assets (Weeks 1–4)

**Goal**: Define narrative, refresh materials, showcase existing work.

**Plans**: 4

- [ ] 01-01: Craft "AI Automation Engineer" personal brand statement
- [ ] 01-02: Update GitHub: READMEs, pinned repos, showcase agent-core, automation-engine, trading-lab
- [ ] 01-03: Build portfolio site highlighting projects with metrics (stars, CI, features)
- [ ] 01-04: Write 3 technical blog posts (deep dives on GSD, agent-core architecture, automation patterns)

### Phase 2: Job Acquisition (Weeks 5–12)

**Goal**: Secure full-time AI/automation engineering role ($150k+ TC).

**Plans**: 4

- [ ] 02-01: Identify 50 target companies (AI startups, infra, quant, remote-friendly)
- [ ] 02-02: Tailor resume/LinkedIn to highlight outcomes (repos built, GSD process, deployment)
- [ ] 02-03: Systematic outreach (2/week to founders/eng managers) with value-first messages
- [ ] 02-04: Interview prep: LeetCode (Python), system design, architecture deep dives (prepare agent-core demo)

### Phase 3: Consulting Launch (Weeks 13–20)

**Goal**: Land first consulting contracts ($10k/mo active revenue).

**Plans**: 4

- [ ] 03-01: Define offer: "AI Automation Audit & Implementation" ($5k–$15k/project)
- [ ] 03-02: Create sales materials (one-pager, case studies from agent-core/automation-engine)
- [ ] 03-03: Reach out to previous contacts, post on LinkedIn, Twitter about availability
- [ ] 03-04: Deliver first project with GSD process; collect testimonial; systematize delivery (templates)

### Phase 4: Productization (Weeks 21–32)

**Goal**: Turn core repos into revenue-generating products (SaaS/API).

**Plans**: 4

- [ ] 04-01: Choose MVP: automation-engine SaaS (hosted scheduler + webhook) or agent-core API
- [ ] 04-02: Build multi-tenant architecture (usage billing, accounts)
- [ ] 04-03: Implement Stripe billing (subscriptions, usage caps)
- [ ] 04-04: Launch public beta; acquire 10 paying customers ($500/mo ARR)

### Phase 5: Passive Scale (Weeks 33–52)

**Goal**: Diversify into content, affiliate, managed services ($5k/mo passive).

**Plans**: 4

- [ ] 05-01: Create comprehensive video course: "Build AI Agents with GSD" ($297)
- [ ] 05-02: Launch affiliate program for OpenClaw/agent-core (15% rev share)
- [03-03]: Write and monetize high-value Substack: "AI Automation Weekly"
- [ ] 05-04: Explore managed hosting: "We run your automation-engine" ($299/mo)

---

**Total plans**: 20

**Milestones**:
- Month 3: Job offer received OR $5k consulting revenue
- Month 6: Consulting $10k/mo OR product MRR $2k
- Month 12: $15k/mo total income (job + active + passive)
