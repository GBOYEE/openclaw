# Roadmap: Grants

## Phases

- [ ] **Phase 1: Research** — Identify suitable programs, deadlines, eligibility
- [ ] **Phase 2: Materials** — Draft boilerplate, CV, project descriptions, budget templates
- [ ] **Phase 3: Proposal** — Write and review full proposals for top targets
- [ ] **Phase 4: Submit & Follow-up** — Final review, submission, post-submission interactions

### Phase 1 (v0.5.0)

Plans:
- 01-01: List top 10 grant programs with deadlines, amounts, fit
- 01-02: Register accounts on funding portals (EU, NSF, etc.)
- 01-03: Gather required documents (CV, organizational info, bank details)
- 01-04: Note: Potential partners (universities, research institutes) for collaboration

### Phase 2 (v0.6.0)

Plans:
- 02-01: Write boilerplate project description (OpenClaw stack, impact)
- 02-02: Prepare CV/biosketch (highlight GSD, repos, achievements)
- 02-03: Create budget template (personnel, equipment, travel, overhead)
- 02-04: Collect letters of intent/partnership if needed

### Phase 3 (v0.7.0)

Plans:
- 03-01: Tailor proposal for first target (e.g., Horizon Europe "Open Source AI")
- 03-02: Internal review (self + perhaps mentor/peer)
- 03-03: Revise based on feedback
- 03-04: Prepare attachments (CVs, diagrams, ethics checklist)

### Phase 4 (v0.8.0 → v1.0.0)

Plans:
- 04-01: Final quality check (page limits, formatting)
- 04-02: Submit via portal before deadline
- 04-03: Log submission confirmation; track status
- 04-04: Follow up if requested; prepare for interview/presentation

**Total:** 16 plans

**Timeframe:** 6–9 months (part-time alongside consulting/job)
